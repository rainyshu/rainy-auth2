# Isme Nest Server

## 简介

本项目是 [Vue Naive Admin 2.0](https://github.com/zclzone/vue-naive-admin) 的后端服务，使用 Nestjs + TypeOrm + MySql + Redis 搭建，实现了 JWT 认证、菜单管理、RBAC 权限控制核心等功能。

## 预览

[https://admin.isme.top](https://admin.isme.top)
